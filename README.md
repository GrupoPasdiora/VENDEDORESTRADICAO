# Metas do Dia — Tradição Cadu (Dashboard Vendedores)

## Deploy no GitHub Pages

1. Crie um repositório **privado** (ex: `metas-tradicao`)
2. Suba os arquivos:
```bash
git init && git add . && git commit -m "Metas vendedores v1"
git branch -M main
git remote add origin git@github.com:SEU-USUARIO/metas-tradicao.git
git push -u origin main
```
3. Settings → Pages → Branch: main → Save
4. URL: `https://SEU-USUARIO.github.io/metas-tradicao/`

Compartilhe a URL com os vendedores. Cada um seleciona seu nome.
